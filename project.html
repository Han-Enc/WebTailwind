Personal Github

웹 코드의 관리

웹 페이지 개발을 꾸준히 하면서 관리를 쉽게 하기 위해서는

github 계정

이 필수적이라고 할 수 있다. github는 단순히 자신이 짠 코드들을 프로젝트 별로 모아볼 수 있을 뿐만 아니라 상대에게 자신의 코드를 공유하기도 쉽고, 어떠한 히스토리를 거쳐왔는지 확인하기도 쉬우며, 날짜 정리는 물론 자신이 언제 개발을 진행했는지도 확인할 수 있다. 내 코드를 알리는 것만큼 상대의 코드를 아는 것도 쉽다. 댓글을 달며 서로 의견 공유를 할 수도 있다. 개발자는 자신의 코드에 설명과 웹페이지 링크 등을 첨부할 수 있으며, 코드를 수정할 때 어느 부분을 수정했는지도 기재할 수 있다.
